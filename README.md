# Playlist Management System

A console-based application developed in C that allows users to create, manage, and browse music playlists. The application supports adding and removing songs, sorting playlists by rating, and deleting unwanted playlists. It was developed as a university project to practice procedural programming, file handling, and dynamic memory management.

## Features

- Create custom music playlists.
- Add and remove songs from playlists.
- Display playlists sorted by rating (descending).
- Delete existing playlists.

## Project Structure

- `main.c` – Main source code of the application.
- `playlisturi.txt` – Stores information about all playlists.
- `playlist1.txt`, `playlist2.txt`, `playlist3.txt` – Example playlist files.

## Technologies

- **Language:** C
- **Interface:** Console application

## Getting Started

### Using the terminal

Clone the repository:

```bash
git clone https://github.com/onica-lorena/playlist-management-system.git
```

Compile the project:

```bash
gcc main.c -o playlist-management-system
```

Run the application:

```bash
./playlist-management-system
```

### Using an IDE

You can also open the project in:

- Code::Blocks
- Dev-C++
- Visual Studio

Compile and run the `main.c` file.
